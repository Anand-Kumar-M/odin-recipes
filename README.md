Odin Recipe: HTML Structure Practice
A simple website designed to showcase basic web development skills, specifically focusing on HTML sturucture and site navigation.

Purpose:
The aim of this project is to practice organizing web pages using HTML 5 by building a recipe index.

Feature:
    A home page containing a list of recipes.
    Recipe links that connect the index to individual recipe detail page.
    For each receipe used a mix of ordered lists (steps), unordered lists (ingredients) with appropriate headings.

Concept Demonstrated:
    Web Page structure
    Usage of Links & Paths
